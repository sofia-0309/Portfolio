Sparty Boots Interactive game

An interactive game where the goal of a game level is to create a simple circuit using logic gates that will correctly kick the correct products from the belt based on the instructions presented in the scoreboard.

Game has 8 levels with different instructions, where difficulty increases by level. The game has multiple components like sensors, conveyor belt and sparty which all work together to make a functional game.
