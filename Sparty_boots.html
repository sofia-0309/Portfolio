Sparty Boots Interactive Game

An interactive game where the goal of each level is to create a simple circuit using logic gates to correctly kick the right products from the conveyor belt based on the scoreboard instructions.

The game has 8 levels with increasing difficulty. Components like sensors, conveyor belts, and Sparty work together to create a functional and engaging game experience.

Technology Stack

  • Language: C++
  • Framework: WxWidget
  • Version Control: Git
  • Design Principles: OOP Concepts
