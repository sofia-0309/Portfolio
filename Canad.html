Sparty in the Box and Integration

A creative project where I implemented and integrated Sparty In A Box machines into the larger virtual environment. This project involved designing, building, and animating machines, as well as implementing the adapter pattern to adapt the machine into a previously developed environment.

Development Process:

Phase 1: Developed an additional prop and designed the initial Sparty In A Box machine.
Phase 2: Implemented machine functionality, integrated it into the Canadian Experience system, and produced a creative movie.

Technology Stack

  • C++
  • WxWidgets
  • WxFormBuilder
  • Visual Paradigm for UML design
